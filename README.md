## 实体模型解析

## 常用Dto类型
- UpdateDto
  - 字段可空
- FilterDto
  - 有索引
  - 不为空
  - 创建时间（最大最小值）
- ItemDto 作为数组中的元素时
  - 过滤Content字段
  - 过滤超过1000长度的字段
  - 过滤List字段
  - 导航字段处理
  - **字段可空要与原类型保持一致**
- ShortDto 
  - 过滤Content字段
  - 过滤超过2000长度的字段
  - 导航字段处理
